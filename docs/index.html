Chaos proxy for LLM apps

Break the provider.
Not your app.

A drop-in proxy for OpenAI-compatible APIs that injects failures on purpose — latency spikes, 4xx/5xx errors, rate limits — so your retries and fallbacks get tested before production does it for you.

pip install agentbreak View source
How it works
your app
→
agentbreak
→
chaos or provider

Mock mode

No real API. Returns fake responses and injected errors. Zero cost, works fully offline.

app → agentbreak → mock

Proxy mode

Forwards to your real provider and randomly injects latency spikes, 429s, and 5xx errors.

app → agentbreak → provider
Quick start
bash · Python 3.10+ 
# 1. install
pip install agentbreak

# 2. start in mock mode — no API key required
agentbreak start \
  --mode mock \
  --scenario mixed-transient \
  --fail-rate 0.2

# 3. point your LLM client at agentbreak
export OPENAI_BASE_URL=http://localhost:5000/v1
export OPENAI_API_KEY=dummy

After a run, hit /_agentbreak/scorecard to review retries, silent failures, and idempotency stats.
