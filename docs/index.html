AgentBreak

AgentBreak is a minimal chaos proxy for OpenAI-compatible chat completions and MCP agent runtimes.

Use agentbreak inspect to discover MCP tools, resources, and prompts, then agentbreak serve to expose the mirrored MCP endpoint and chat endpoint.

Quick Start

cp config.example.yaml application.yaml
cp scenarios.example.yaml scenarios.yaml
agentbreak serve --config application.yaml --scenarios scenarios.yaml

Endpoints:

  • POST /v1/chat/completions
  • POST /mcp

MCP Inspect

agentbreak inspect --config application.yaml
agentbreak serve --config application.yaml --scenarios scenarios.yaml

This writes .agentbreak/registry.json and uses it to mirror discovered tools, resources, and prompts.

Current Scope

  • Targets: llm_chat, mcp_tool
  • MCP methods: tools/list, tools/call, resources/list, resources/read, prompts/list, prompts/get
  • Faults: http_error, latency, timeout, empty_response, invalid_json, schema_violation, wrong_content, large_response

Deferred targets are documented in TODO_SCENARIOS.md.

Verification

agentbreak verify

Runs the local pytest verification suite.
